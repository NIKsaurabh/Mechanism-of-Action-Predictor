Important Instructions

File Upload

File being uploaded should be a comma seperated value file with 876 columns and any number of rows.
First column should be ID value.
Second column should be cp_type.
Third column should be cp_time.
Fourth column should be cp_dose.
5th to 776th columns should be gene columns (ex- g-0, g-1,..., g-772).
777th to 876th columns should be cell columns (ex- c-0, c-1,..., c-99).

Download sample upload file

Predict and Download

After uploading the file click on "Predict and Download" button.
It will take time according to number of data points to predict and
it will download a zip file containing two csv files:
1. File containing predictions.
2. File containing probability of each predictions
